## 3. ROS2 节点和话题的基本理解

### 3.1 节点 node

节点可以理解为 ROS2 系统中的一个运行程序。

例如：

```text
相机驱动节点
雷达节点
机械臂控制节点
目标识别节点
底盘控制节点
```

查看节点：

```bash
ros2 node list
```

---

### 3.2 话题 topic

话题可以理解为 ROS2 系统中的消息频道。

一个节点可以向某个话题发布消息，另一个节点可以订阅这个话题接收消息。

例如：

```text
相机节点发布图像到 /gemini_camera/rgb/image_raw
识别程序订阅图像话题进行目标识别
控制程序发布命令到 /servo_controller
机械臂控制节点订阅命令并控制舵机
```

查看话题：

```bash
ros2 topic list
```

查看话题类型：

```bash
ros2 topic info /话题名
```

查看详细发布者和订阅者：

```bash
ros2 topic info -v /话题名
```
