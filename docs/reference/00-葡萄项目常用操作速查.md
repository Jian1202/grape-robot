# ROSLander 葡萄项目常用操作速查

## 1. SSH 连接机器人

优先使用主机名：

```bash
ssh ubuntu@ubuntu.local
```

如果主机名不可用，先在机器人上查看 IP：

```bash
hostname -I
```

或：

```bash
nmcli device status
nmcli device show
```

然后连接：

```bash
ssh ubuntu@机器人IP
```

---

## 2. NoMachine 连接

```text
Host：ubuntu.local
Port：4000
Protocol：NX
Username：ubuntu
```

如果 `ubuntu.local` 不可用，改用机器人当前局域网 IP。

---

## 3. 测试机器人网络

Mac 终端：

```bash
ping ubuntu.local
```

或：

```bash
ping 机器人IP
```

---

## 4. 一键运行 YOLO 视觉检测

必须在 NoMachine 图形桌面的终端执行：

```bash
~/teams/ctrlteam/grape_robot/scripts/run_vision.sh
```

只检测未成熟葡萄：

```bash
TARGET_CLASS=unripe_grape \
~/teams/ctrlteam/grape_robot/scripts/run_vision.sh
```

临时修改置信度：

```bash
CONFIDENCE=0.5 \
~/teams/ctrlteam/grape_robot/scripts/run_vision.sh
```

退出：

```text
图像窗口按 q 或 Esc
终端按 Ctrl+C
```

## 8. 上传新 YOLO 模型

Mac 终端：

```bash
scp "/本地路径/best.pt" \
ubuntu@ubuntu.local:/tmp/best.pt
```

机器人终端保存新模型：

```bash
PROJECT=~/teams/ctrlteam/grape_robot
MODEL_NAME="grape_$(date +%Y%m%d_%H%M%S).pt"

cp /tmp/best.pt \
"$PROJECT/models/archive/$MODEL_NAME"
```

检查模型能否加载：

```bash
python3 - <<PY
from ultralytics import YOLO

path = "$HOME/teams/ctrlteam/grape_robot/models/archive/$MODEL_NAME"
model = YOLO(path)

print("MODEL_OK")
print("task:", model.task)
print("classes:", model.names)
PY
```

切换当前模型：

```bash
ln -sfn "archive/$MODEL_NAME" \
"$PROJECT/models/current.pt"
```

查看当前模型：

```bash
readlink -f \
~/teams/ctrlteam/grape_robot/models/current.pt
```

## 10. 手动同步代码到 ROS2 工作区

通常由 `run_vision.sh` 自动完成。

手动同步：

```bash
PROJECT=~/teams/ctrlteam/grape_robot
SRC_DIR=~/ros2_ws/src/example/example/rgbd_function

cp "$PROJECT/code/track_and_grab.py" \
"$SRC_DIR/track_and_grab.py"

cp "$PROJECT/launch/track_and_grab.launch.py" \
"$SRC_DIR/track_and_grab.launch.py"
```

---

## 11. 手动检查并编译

```bash
SRC_DIR=~/ros2_ws/src/example/example/rgbd_function

python3 -m py_compile \
"$SRC_DIR/track_and_grab.py" \
"$SRC_DIR/track_and_grab.launch.py"

cd ~/ros2_ws
colcon build --packages-select example --symlink-install
source ~/ros2_ws/install/setup.zsh
```

---

## 12. 手动启动 YOLO 安全视觉模式

```bash
sudo systemctl stop start_app_node.service

source ~/ros2_ws/install/setup.zsh
export need_compile=False

ros2 launch example track_and_grab.launch.py \
  detector:=yolo \
  model_path:=/home/ubuntu/teams/ctrlteam/grape_robot/models/current.pt \
  target_class:=ripe_grape \
  confidence:=0.4 \
  imgsz:=320 \
  enable_arm:=false \
  start:=true
```

## 17. 当前安全要求

当前只使用：

```text
enable_arm=false
```

在没有单独加入“允许跟踪、禁止抓取”的安全开关前，不要直接启用机械臂。
