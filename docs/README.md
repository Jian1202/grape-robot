# 项目文档入口

本目录集中保存葡萄机器人项目的技术说明、问题分析、原始参考资料和阶段报告。

## 建议阅读顺序

1. [项目总览与技术路线](reference/01-项目总览与技术路线.md)
2. [葡萄采摘问题分析与实施建议](reports/2026-07-13-葡萄采摘问题分析与实施建议.md)
3. [精选原始资料索引](source-materials/README.md)
4. [机械臂控制接口](reference/04-机械臂控制接口.md)
5. [机械臂动作组文件](reference/05-机械臂动作组文件.md)
6. [YOLO 接入与运行维护](reference/08-YOLO接入与运行维护.md)

## 目录说明

- `reference/`：机器人、ROS2、机械臂、YOLO 等长期参考文档。
- `reports/`：阶段分析、问题复盘和实施建议。
- `source-materials/`：从 ROSLander 教学资料中精选的原始手册和课程文件。

## 使用约定

- 原始资料用于理解厂商方案和确认硬件约束，不代表当前 ROS2 仓库已经实现其中功能。
- `source-materials/` 中不少资料基于 ROS1，不能直接覆盖当前 ROS2 代码。
- 涉及机械臂和底盘的实验必须分阶段启用，先验证反馈与安全范围，再进行自动抓取。
